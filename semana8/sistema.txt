1. Información que muestra cada comando:
   - uname -a: Muestra información detallada del sistema operativo, incluyendo el nombre del kernel de Linux, el hostname de la máquina, la versión del kernel, la fecha de compilación y la arquitectura del procesador (por ejemplo, x86_64 o arm).
   - whoami: Muestra el nombre de usuario exacto de la sesión activa que está ejecutando los comandos en la terminal.

2. Utilidad al administrar un servidor:
   Conocer esta información es vital para un administrador por varias razones:
   - Compatibilidad: Permite saber con precisión qué arquitectura y versión del sistema operativo se están ejecutando para instalar las dependencias o paquetes de software adecuados.
   - Seguridad: Ayuda a identificar si el kernel del sistema está desactualizado o expuesto a vulnerabilidades conocidas que requieran parches urgentes.
   - Control de permisos: "whoami" permite verificar instantáneamente si se está trabajando con un usuario con privilegios restringidos o si se tiene acceso de superusuario (root), evitando ejecutar comandos peligrosos de forma accidental.
